# Trabalho-Pratico-Estacoes-Meteorologicas
Trabalho prático final do curso de Programação e Desenvolvimento de Software I, da Univerdade Federal de Minas Gerais, ministrado pelo professor Pedro Henrique Barros. 
O trabalho foi escrito manualmente, gastando aproximadamente 15 horas de trabalho. Documentações e fóruns online foram consultados, e algumas funcionalidades externas ao funcionamento bruto do código foram importadas de terceiros*.

Sobre o programa: 
 O Programa se consiste em um painel de controle referente à um sistema de Estações Meteorológicas. Por meio dele, é possível registrar e configurar estações, inserir leituras destas, vizualizar a lista de estações existentes, calcular estatísticas relevantes, detectar medidas anomalas, exportar e importar os dados em .csv.


*as funcionalidades de Makefile foram feitas com ferramentas de I.A, e as funções clean() e clear(); foram retiradas de fóruns online.
